# 🌟 my_website
我的个人站点，细品前端工程化技术栈

## 🚀 项目特色

✅ 基于VitePress的现代化文档架构  
✅ 集成Unocss原子化CSS方案  
✅ 支持Markdown增强语法与自定义组件  
✅ 自动化构建部署工作流
✅ 部署GitHub Pages，请参阅[VitePress部署指南](https://vitepress.dev/zh/guide/deploy#github-pages)

## 🛠 技术栈全景
| 分类         | 技术选型                          |
|-------------|-----------------------------------|
| 核心框架     | VitePress 1.6.3                   |
| 样式方案     | UnoCSS + PostCSS                  |
| 构建工具     | Vite 5.x                          |
| 包管理       | pnpm 8.x                          |
| 持续集成     | GitHub Actions                    |

## 📂 项目结构
```
my_website/
├── docs/                 # 文档核心目录
│   ├── .vitepress/      # 主题配置
│   ├── public/          # 静态资源
│   └── src/             # 内容模块
└── package.json         # 依赖管理
```

## ⚡ 快速开始
```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm run docs:dev
```
